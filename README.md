# 握笔姿势识别矫正系统 (Pen-Grip Posture Recognition & Correction System)

本项目为“大创项目：握笔姿势识别矫正系统”的可视化前端演示原型。

本项目展示了一个基于 AI 分析（YOLO 算法核心与 OpenCV 技术采集）概念的用户握笔姿势识别的可视化界面。当前形态为静态的纯前端原型页面，无需真实 YOLO 模型推断、无需后方服务端运行以及硬件摄像头连接。骨架坐标和推理参数已针对9种常见握笔姿态进行精细的数据 Mock，只需通过按键便可切换不同状态进行演示。

## 功能特性

- **多姿势支持**：涵盖 9 种定义握姿（拇指包笔、翻腕、勾腕、手指过直、四指围笔、握笔过低、食指弯曲、埋头、正确握法），提供对应的骨架与分析说明。
- **高逼真骨架渲染**：骨架关键点直接提取自项目真实 YOLO 测试中输送的 JSON 对象数据，利用 SVG 和 React 实现了精准映射展示。
- **动态仿刷新**：加入了拟真的微小抖动噪点，模拟真实 CV 模块运算时的误差感，让无硬件的 Demo 一样生动自然。
- **响应式 UI**：基于现代 Bento 网格设计构建，适应主要屏幕设备的体验，内置状态指示图表以及流体界面动画提示。

## 快速安装与运行

**前置依赖**：系统中需包含 [Node.js](https://nodejs.org/) 环境。

1. **克隆项目**
   \`\`\`bash
   git clone <仓库地址>
   cd 握笔姿势识别矫正系统
   \`\`\`

2. **安装依赖包**
   本项目使用 \`npm\` 或 \`pnpm\` 管理依赖。
   \`\`\`bash
   npm install
   # 或
   # pnpm install
   \`\`\`

3. **启动本地开发服务器**
   \`\`\`bash
   npm run dev
   # 或
   # pnpm dev
   \`\`\`

4. **查看演示**
   控制台输出端口绑定地址（如 http://localhost:3000/ 或类似地址），在浏览器中打开即可开始体验。
